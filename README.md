## WHMCS disable the auto-renew domain in the order form

This hook for WHMCS allows you to disable the auto-renew domain in the order form.

**Tested on WHMCS Version: 7.4.2**

**WHMCS Template: SIX**

**Order Form Template: Standard Cart**
